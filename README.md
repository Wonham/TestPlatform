# TestPlatform

MATLAB-based LiDAR test data processing platform for autonomous driving research.

## Overview

A comprehensive tool for processing LiDAR point cloud data, supporting:
- Power loss analysis
- Beam angle calibration
- Beam quality assessment (IPR/PSLR)
- Spot quality metrics

## Directory Structure



## Usage

1. Place your test data in `I0.txt`
2. Run `main.m` in MATLAB
3. Check results in `Result/` directory

## Requirements

- MATLAB R2020b or later
